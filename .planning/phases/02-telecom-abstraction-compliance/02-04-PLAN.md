    ```python
    class USComplianceModule(ComplianceModule):
        """US jurisdiction compliance: TCPA + DNC enforcement. D-11, D-12."""

        def __init__(self, consent_db: ConsentDB, dnc_list: DNCList):
            self._consent_db = consent_db
            self._dnc_list = dnc_list

        async def check_outbound(
            self, destination: str, session: TelecomSession
        ) -> ComplianceResult:
            """Run all US compliance checks in order. First failure short-circuits.

            Check order:
            1. DNC list (cheapest — single SQLite lookup)
            2. Time-of-day (no I/O — pure timezone math)
            3. Consent verification (SQLite lookup)
            """
    ```

    Check order rationale: DNC is cheapest (single index lookup). Time-of-day is pure computation (no I/O). Consent requires a query. Short-circuit on first failure to minimize work.

    Import check functions from tcpa.py and dnc_check.py. The module orchestrates them in sequence and returns the first failure or a final pass result.
