<objective>
Extend the LLM client and voice pipeline to handle tool-calling LLM responses.

Purpose: When an LLM decides to invoke a tool (call, sms, hangup, transfer) instead of speaking, the pipeline must intercept the tool call, execute it via ToolExecutor, feed the result back to the LLM, and continue the conversation (D-14, AGENT-01). This is the bridge between the tool definitions (Plan 02) and the actual execution path.

Output: Extended llm.py with tool-call streaming, extended pipeline.py with tool-call interception, unit tests.
</objective>

    tool_calls_accumulator = {}  # index -> {"id": str, "name": str, "arguments": str}
    async for chunk in stream:
        delta = chunk.choices[0].delta
        if delta.content is not None:
            token_count += 1
            yield delta.content
        if hasattr(delta, 'tool_calls') and delta.tool_calls:

                    if tc.function.arguments:

    **Modify `_respond()` method:**
    The current flow is:
    1. STT transcribe
    2. LLM stream tokens -> token_queue
    3. TTS consumes token_queue -> send audio

    The new flow adds tool-call interception:
    1. STT transcribe
    2. LLM stream tokens -> token_queue (with tools if tool_executor is set)
    3. TTS consumes token_queue -> send audio
    4. **NEW:** If ToolCallSentinel received from token_queue, flush TTS, execute tool, feed result back to LLM, loop

    **Modify `feed_tokens()` inner function in _respond():**
    Change the LLM call to pass tools when tool_executor is set:
    ```python
    tools = get_tools() if self.tool_executor else None
    async for token in self.llm.stream_response(transcript, session.history, tools=tools):
        if session._tts_cancel.is_set():
            break
        if isinstance(token, ToolCallSentinel):
            # Put sentinel in queue for TTS loop to see, then stop feeding
            await token_queue.put(token)
            return  # Don't put None sentinel — tool handling continues below
        await token_queue.put(token)
        full_response.append(token)
    await token_queue.put(None)  # Normal end sentinel
    ```

    **Modify TTS consumption loop in _respond():**
    After the TTS stream loop, check if the last item from the queue was a ToolCallSentinel. The cleanest approach: have the TTS loop break on sentinel.

    Restructure `_respond()` to support a tool-call retry loop:
    ```python
    async def _respond(self, session, audio_16k, send_audio_callback):
        turn_start = time.monotonic()
        try:
            # STT
            segments = await self.stt.transcribe_buffer(audio_16k)
            transcript = " ".join(segments).strip()
            if not transcript:
                session.vad.set_pipeline_state(PipelineState.LISTENING)
                return

            # Tool-call loop: LLM may return tool calls that require re-prompting
            max_tool_rounds = 5  # Prevent infinite loops
            messages_for_round = transcript  # First round uses transcript
            extra_history = []  # Tool results to append

            for round_num in range(max_tool_rounds):
                session.vad.set_pipeline_state(PipelineState.SPEAKING)
                session._tts_cancel.clear()

                token_queue = asyncio.Queue()
                full_response = []
                tool_sentinel = None

                tools = get_tools() if self.tool_executor else None
                history = session.history + extra_history

                async def feed_tokens():
                    nonlocal tool_sentinel
                    async for token in self.llm.stream_response(messages_for_round, history, tools=tools):
                        if session._tts_cancel.is_set():
                            break
                        if isinstance(token, ToolCallSentinel):
                            tool_sentinel = token
                            await token_queue.put(None)  # Signal TTS to stop
                            return
                        await token_queue.put(token)
                        full_response.append(token)
                    await token_queue.put(None)

                llm_task = asyncio.create_task(feed_tokens())

                # TTS streaming
                first_audio = True
                async for samples, sample_rate in self.tts.synthesize_stream(token_queue):
                    if session._tts_cancel.is_set():
                        break
                    if first_audio:
                        ttfa = time.monotonic()
                        logger.info("pipeline.first_audio", call_uuid=session.call_uuid, total_latency_ms=round((ttfa - turn_start) * 1000))
                        first_audio = False
                    pcm_bytes = downsample_24k_to_8k(samples)
                    await send_audio_callback(pcm_bytes)

                await llm_task

                if tool_sentinel and self.tool_executor:
                    # Execute tool calls
                    import json
                    tool_results = []
                    for tc in tool_sentinel.tool_calls:
                        args = json.loads(tc["arguments"]) if isinstance(tc["arguments"], str) else tc["arguments"]
                        logger.info("pipeline.tool_call", call_uuid=session.call_uuid, tool=tc["name"], args=args)
                        result = await self.tool_executor.execute(tc["name"], args, session)
                        logger.info("pipeline.tool_result", call_uuid=session.call_uuid, tool=tc["name"], result=result)
                        tool_results.append({"tool_call_id": tc["id"], "result": result})

                    # Build history for next LLM round
                    # Add assistant message with tool_calls
                    extra_history.append({"role": "assistant", "content": None, "tool_calls": [
                        {"id": tc["id"], "type": "function", "function": {"name": tc["name"], "arguments": tc["arguments"]}}
                        for tc in tool_sentinel.tool_calls
                    ]})
                    # Add tool results
                    for tr in tool_results:
                        extra_history.append(self.llm.build_tool_result_entry(tr["tool_call_id"], tr["result"]))

                    messages_for_round = ""  # Empty — LLM continues from tool results
                    tool_sentinel = None
                    continue  # Next round
                else:
                    # Normal text response — done
                    break

            # Update conversation history
            session.history.append({"role": "user", "content": transcript})
            if full_response:
                session.history.append({"role": "assistant", "content": "".join(full_response)})

            turn_end = time.monotonic()
            logger.info("pipeline.turn_complete", call_uuid=session.call_uuid, total_ms=round((turn_end - turn_start) * 1000))

        except asyncio.CancelledError:
            logger.info("pipeline.turn_cancelled", call_uuid=session.call_uuid)
        except Exception as e:
            logger.error("pipeline.turn_error", call_uuid=session.call_uuid, error=str(e))
        finally:
            session.vad.set_pipeline_state(PipelineState.LISTENING)
    ```

    **Key design decisions:**
    - Tool calls flush TTS (via None sentinel to queue) before executing — prevents pipeline deadlock (Pitfall 2)
    - max_tool_rounds=5 prevents infinite tool-call loops
    - Tool results are added to extra_history, not session.history, until the turn completes
    - When tool_executor is None, pipeline behaves exactly as before (backward compat)
