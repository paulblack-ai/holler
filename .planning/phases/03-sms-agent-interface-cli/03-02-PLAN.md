         - additionalProperties: false, strict: true

    - `ToolCallSentinel` dataclass:

      ```python
      @dataclass
      class ToolCallSentinel:
          tool_calls: list  # List of {"id": str, "name": str, "arguments": str (JSON)}
      ```
      This sentinel is yielded by LLMClient.stream_response() when the LLM invokes a tool instead of producing text. The pipeline coordinator intercepts it.

    **holler/core/agent/executor.py** (per D-02, D-03):
    - `ToolExecutor` class:
      - `__init__(self, esl, sms_client, compliance_gateway, pool)`: stores references. Use TYPE_CHECKING for type hints.
      - `async execute(self, tool_name: str, arguments: dict, session) -> dict`:
        - try/except ComplianceBlockError -> return `{"status": "blocked", "reason": str(e)}` (D-02)
        - try/except Exception -> return `{"status": "error", "reason": str(e)}`
        - Dispatch by name:
          - "call" -> `_execute_call(arguments, session)`
          - "sms" -> `_execute_sms(arguments, session)`
          - "hangup" -> `_execute_hangup(arguments, session)`
          - "transfer" -> `_execute_transfer(arguments, session)`
          - else -> `{"status": "error", "reason": f"unknown_tool: {tool_name}"}`

      - `async _execute_call(self, args, session) -> dict`:
        - destination = args["destination"]
        - call_uuid = await self._gateway.originate_checked(self._esl, self._pool, session)
        - return `{"status": "ok", "call_uuid": call_uuid}`

      - `async _execute_sms(self, args, session) -> dict`:
        - destination = args["destination"], message = args["message"]
        - log_id = str(uuid.uuid4())
        - session_for_sms = create a minimal TelecomSession(destination=destination, did=session.did, ...)
        - await self._gateway.sms_checked(self._sms, self._pool, session_for_sms, message, log_id)
        - return `{"status": "ok", "message_id": log_id}`

      - `async _execute_hangup(self, args, session) -> dict`:
        - call_uuid = args["call_uuid"]
        - await self._esl.hangup(call_uuid)
        - return `{"status": "ok"}`

      - `async _execute_transfer(self, args, session) -> dict` (per D-03):
        - call_uuid = args["call_uuid"], destination = args["destination"]
        - Run compliance check on destination first: `module = self._gateway._router.resolve(destination)` then `result = await module.check_outbound(destination, session)`. If not passed, raise ComplianceBlockError.
        - ESL transfer: `cmd = f"api uuid_transfer {call_uuid} sofia/gateway/sip_trunk/{destination}"`
        - `await self._esl.send_raw(cmd)`
        - return `{"status": "ok", "transferred_to": destination}`

    - Use structlog for logging. Use TYPE_CHECKING guard for all Holler imports.

    **holler/core/agent/adapters.py** (per D-04):
    - `def openai_to_anthropic(openai_tool: dict) -> dict`:
      - Extract `fn = openai_tool["function"]`
      - Return `{"name": fn["name"], "description": fn["description"], "input_schema": fn["parameters"]}`
      - (Drops "type", "strict" — Anthropic doesn't use them)

    - `def anthropic_response_to_tool_calls(content_blocks: list) -> list`:
      - Return `[{"id": b["id"], "name": b["name"], "arguments": b["input"]} for b in content_blocks if b["type"] == "tool_use"]`

    - `def openai_tools_to_anthropic(tools: list) -> list`:
      - Return `[openai_to_anthropic(t) for t in tools]`
