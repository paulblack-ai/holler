    ```python
    async def sms_checked(
        self,
        sms_client: "SMSClient",
        pool: "NumberPool",
        session: "TelecomSession",
        message: str,
        log_id: str,
    ) -> str:
    ```

    The method must:
    1. Resolve the compliance module via `self._router.resolve(session.destination)`
    2. Run `module.check_outbound(session.destination, session)` with `asyncio.wait_for(timeout=self._timeout)` — same fail-closed pattern as originate_checked()
    3. Catch `NoComplianceModuleError`, `asyncio.TimeoutError`, and `Exception` — all produce deny `ComplianceResult`
    4. Always write audit log entry via `self._audit.write()` with `channel: "sms"`, message_log_id, destination, result, reason
    5. Set `session.compliance_result = result`
    6. If `not result.passed`: release DID via `pool.release(session.did)`, raise `ComplianceBlockError(result.reason)`
    7. If passed: call `await sms_client.send(session.destination, message, log_id)`, return `log_id`
